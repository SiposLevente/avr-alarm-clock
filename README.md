# AVR Alarm clock
This project aims to create an alarm clock.

## Planned Features:
* Displays the time on a 4-digit 7-segment display.
* Adjustable time with the use of buttons.
* 9 settable alarms.
* Adjustable brightness. Can be set to automatic brightness control.
